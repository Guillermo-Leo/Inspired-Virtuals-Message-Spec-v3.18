Cricket

Event codes

Cricket is event code 17

Additional event card attributes

In the event card the following attributes are added to the event element:

  • MeetingName â€“ meeting name.
  • Team1 - Name of the first team to bat (e.g. "South Africa").
  • Team1ShortName - Shorter version of the first team's name (e.g. "RSA").
  • Team2 - Name of the second team to bat (e.g. "India").
  • Team2ShortName - Shorter version of the second team's name (e.g. IND").

Additional result attributes

There are no additional attributes or elements in the result message
