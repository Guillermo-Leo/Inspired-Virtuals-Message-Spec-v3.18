Matchday

Event codes

Matchday is event code 11 (Matchday), 16 (Matchday Light), 38 (4 goal Soccer 3 Matchday), 37 (6 goal Soccer 3 Matchday) or 39 (8 goal Soccer 3 Matchday)

Description

Matchday is a collection of 8 soccer events matches that happen simultaniously. There are optional aggregate markets that allows bets on the total number of goals in all 8 matches. For each matchday there are 8 soccer events event cards and results sent and an additional matchday card and result sent only if some aggregate markets are enabled.

Additional event card attributes

In the event card the following attributes are added to the event element:

  • EventName - name of tournament.
  • IsLeague â€“ 0 if this is a standalone matchday, 1 if this matchday is part of a league.

If the IsLeague attribute is set to 1 the following attributes are also added to the event element:

  • LeagueName - Name of the league the MatchDay event belongs to
  • LeagueMatchdayID - ID of the league the MatchDay event belongs to

There is also a matches sub-element that contains 8 match sub elements. Each match sub element contains the following attributes:

  • MatchdayIndex - a 1 based index for the match.
  • EventId - the event Id of the match that is part of the matchday.

Additional result attributes

There are no additional attributes for the result.
