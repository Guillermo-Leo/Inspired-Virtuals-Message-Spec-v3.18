Result

The result message contains the outcome of the event and result of each market and any partial markets. It is sent once the event has been fully played out and seen by anyone who has placed a bet so the BMS should be able to immediately display the result and pay out any winning bets. The exception to this is if there is any streaming delay in the delivery of the event visuals which could mean this message is sent a few seconds before the final result is seen.

Each "Message" element has an "event" sub-element

Each "event" element has attributes that are common to all event types. These common attributes are:

  • EventId – an integer that uniquely identifies the event.
  • EventGUID – a GUID that uniquely identifies the event.
  • EventType - an integer that uniquely identifies the event type.
  • EventTime - the time the event is due to start.
  • xsi:type - A name that corresponds to the XSD that this message will conform to.

Each Event element contains a number of Market sub-elements containing the following attributes:

  • Type – bet type name, configurable to a platform specific string.
  • BetCode – the numeric code uniquely assigned to this particular type of bet.
  • Price - outcome price. Fractional, Decimal or American depending on OddsType.
  • Result - The index of the winning selection in this market. This corresponds to an index in the event card message

There may also be event specific attributes or elements. These are described in each event type's event specific page

Each Event element may contain a number of PartialMarket sub-elements containing the following attributes:

  • Type – bet type name, configurable to a platform specific string.
  • BetCode – the numeric code uniquely assigned to this particular type of bet.

Each PartialMarket element contains a number of Selection sub-elements, each representing a possible outcome for this market and containing the following attributes:

  • Num - outcome index. This will correspond to an index in the event card message.
  • Result - will list whether a bet on this outcome has won ("Wins) or lost ("Loses"). If the outcome has won, the following optional attribute will also be present:
  • Price - outcome price. Fractional, Decimal or American depending on OddsType.

There may also be event specific attributes or elements. These are described in each event type's event specific page
