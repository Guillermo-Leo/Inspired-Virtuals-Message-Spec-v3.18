CounterStrike

Event codes

CounterStrike is event code 70

Additional event card attributes

There are no additional attributes or elements in the event card message

Additional result attributes

Under normal behaviour, the event element provides an additional "Result" child-element that contains the follow attributes:

  • Winner - The winning team ("Counter-Terrorists" or "Terrorists").
  • WinType - The method of victory("Bomb Defused", "Bomb Exploded", "Counter-Terrorists Win By Elimination" or "Terrorists Win By Elimination").
  • TerroristsRemaining - The number of terrorists remaining at the end of the event.
  • CounterTerroristsRemaining - The number of counter-terrorists remaining at the end of the event.
  • FirstKillTime - Whether the first kill occurred before or after the first kill line ("Before Line" or "After Line").
