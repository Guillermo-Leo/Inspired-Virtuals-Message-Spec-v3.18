Soccer

Event codes

Soccer is event code 41 (4 goal Soccer 3), 40 (6 goal Soccer 3), 42 (8 goal Soccer 3), 9 (Rush 2) or 4 (Platinum)

Additional event card attributes

In the event card the following attributes are added to the event element:

  • Team1 - name of home team.
  • Team2 - name of away team.
  • EventName - name of tournament.
  • Stadium – name of the stadium.
  • AwayTeamUsesAwayKit – 0 if away team will play in their home kit, 1 if they play in their away kit. (Not applicable to Soccer 3)
  • HomeKitId - ID of the kit that the home team is wearing (Soccer 3 only)
  • AwayKitId - ID of the kit that the home team is wearing (Soccer 3 only)
  • MaxClipsInMatch - The maximum number of clips that will be displayed in the match (Soccer 3 only)

If the match is part of a matchday event there are additional attributes:

  • MatchdayID - The Event Id of the matchday
  • MatchdayIndex - The Index of this event within the matchday
  • IsPartOfLeague - 0 if this is a standalone matchday, 1 if this matchday is part of a league.

If the IsLeague attribute is set to 1 the following attributes are also added to the event element:

  • LeagueName - Name of the league the MatchDay event belongs to
  • LeagueMatchdayID - ID of the league the MatchDay event belongs to

Each Selection element for the Asian Handicap market contains an additional attribute:

  • Handicap - a string containing the handicap value specific to that outcome.

Additional result attributes

Under normal behaviour, the event element provides an additional "Result" child-element that contains the follow attributes:

  • Result - score at full time e.g. "2-0"
  • HalfTimeResult - score at half time e.g. "1-0"
  • FirstScorer - String with the position and team of the first player to score (E.g. "Away Centre Forward")

The Market element for the Asian Handicap market contains additional attributes:

  • OutcomeHome - Outcome for home team, see below table.
  • PayoutHome - Payout price for home team. Fractional or Decimal depending on market specific OddsType.
  • OutcomeAway - Outcome for away team, see below table.
  • PayoutAway - Payout price for away team. Fractional or Decimal depending on market specific OddsType.

The market element for the Asian Handicap market does not contain a "Price" attribute.

Possible outcomes for Asian handicap:

Attribute Value Meaning Bet result
W Win Win
P Push Refunded stake
L Lose Lose
WP Half Win 50% Win 50% Refunded stake
LP Half Lose 50% Refunded stake

The double chance market has 2 winning outcomes per event so the result attribute of the Market element will contain 2 winning indexes separated by a comma and the Price attribute will contain 2 corresponding winning prices, separated by a comma
