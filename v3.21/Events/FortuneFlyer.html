Fortune Flyer

Event codes

Fortune Flyer is event code 87.

Additional event card attributes

In the event card the following attributes are added to the event element:

  • MeetingName - meeting name (e.g. "FortuneFlyer").
  • Details - details of the event in a string which can be displayed (e.g. "Fortune Flyer").

Additional result attributes

Under normal behaviour, the event element provides an additional "Result" child-element that contains the follow attributes:

  • Plane1Distance - distance in meters of plane 1 (e.g. "1.3010").
  • Plane2Distance - distance in meters of plane 2 (e.g. "2.5303").
  • Plane3Distance - distance in meters of plane 3 (e.g. "10.9901").
  • Plane4Distance - distance in meters of plane 4 (e.g. "1.3706").
  • TotalDistanceAll - total distance in meters of all planes (e.g. "16.1920").
  • Tricast - order of first 3 planes (e.g. "3,2,4").

Markets and bet codes

The following list includes the most common markets offered, but it may not cover every available option for the event.

Total Distance All Under Over

Betcode: "367"

Outcomes: "X" will be replaced by a number that varies depending on configuration (e.g. 11m).

  • Total distance under Xm
  • Total distance over Xm

Each To Win

Betcode: "368"

Outcomes:

  • Plane 1 wins
  • Plane 2 wins
  • Plane 3 wins
  • Plane 4 wins

Forecast

Betcode: "369"

Outcomes:

Instead of "Selection" elements there will be a "PriceList" sub-element which will contain a comma seperated list of prices. Forecast has prices in the format "12, 13, 14, ..., 1X, 21, 23, 24, ..., 2X, 31, 32, 34, ..., 3X, 41, 42, 43, ..., 4X.....” where 12 is a price for plane 1 coming 1st and plane 2 coming 2nd. The number of prices in the list is (#planes) * (#planes-1), e.g. 4 planes requires 12 prices. There is also a "RTPList" element which contains the associated RTP for each price in a similar comma seperated list.

Tricast

Betcode: "370"

Outcomes:

Instead of "Selection" elements there will be a "PriceList" sub-element which will contain a comma seperated list of prices. Tricast has prices in the format “123, 124, ..., 12X, 132, 134, ..., 13X, 142, 143, ..., 14X.....” where 123 is a price for plane 1 coming 1st and plane 2 coming 2nd and plane 3 coming 3rd. The number of prices in the list is (#planes) * (#planes-1) * (#planes-2), e.g. 4 planes requires 24 prices. There is also a "RTPList" element which contains the associated RTP for each price in a similar comma seperated list.
